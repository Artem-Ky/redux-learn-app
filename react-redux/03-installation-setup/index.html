Интерактив — Минимальное React-Redux приложение

Этот счётчик — полноценное React-Redux приложение, состоящее из трёх частей: Redux store, Provider и React-компонент с хуками. Весь код — ниже.

✔ Живое React-Redux приложение

Шаги установки React-Redux проекта

1
Установить React-Redux

React-Redux — отдельный пакет от redux. Нужны оба.

npm install react-redux redux
2
Создать Redux store

Это уже знакомо из Classic Redux: reducer + createStore.

const store = createStore(rootReducer)
3
Обернуть приложение в Provider

Provider делает store доступным для всех вложенных компонентов.

<Provider store={store}><App /></Provider>
4
Использовать хуки в компонентах

useSelector для чтения данных, useDispatch для отправки действий.

const value = useSelector(state => state.value)

Минимальный код: три шага

Шаг 1: Создать Redux store (уже знакомо из Classic Redux)

import { createStore } from 'redux' interface CounterState { value: number } const initialState: CounterState = { value: 0 } function counterReducer(state = initialState, action: { type: string }) { switch (action.type) { case 'increment': return { value: state.value + 1 } case 'decrement': return { value: state.value - 1 } default: return state } } const store = createStore(counterReducer)

Шаг 2: Обернуть приложение в Provider

import { Provider } from 'react-redux' import { createRoot } from 'react-dom/client' const root = createRoot(document.getElementById('root')!) root.render( <Provider store={store}> <App /> </Provider> )
«The <Provider> component makes the Redux store available to any nested components that need to access the Redux store.»
— react-redux.js.org — Provider
«Компонент <Provider> делает Redux store доступным для любых вложенных компонентов, которым нужен доступ к Redux store.»
— перевод

Шаг 3: Использовать хуки в компонентах

import { useSelector, useDispatch } from 'react-redux' function Counter() { const count = useSelector(state => state.value) const dispatch = useDispatch() return ( <div> <button onClick={() => dispatch({ type: 'decrement' })}>−</button> <span>{count}</span> <button onClick={() => dispatch({ type: 'increment' })}>+</button> </div> ) }
«`useSelector` reads a value from the store state and subscribes to updates, while `useDispatch` returns the store's `dispatch` method to let you dispatch actions.»
— react-redux.js.org — Getting Started
«`useSelector` читает значение из state store и подписывается на обновления, а `useDispatch` возвращает метод `dispatch` store для отправки actions.»
— перевод
Всего три шага:
1. createStore(reducer) — создать store (Classic Redux)
2. <Provider store={store}> — обернуть приложение (React-Redux)
3. useSelector / useDispatch — использовать в компонентах (React-Redux)
