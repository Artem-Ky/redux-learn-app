Теория — Что такое Action?

      

        
Action — обычный JavaScript-объект

        

          Action (экшен) — это простой JavaScript-объект, который имеет поле type.
          Можно думать об экшене как о событии, описывающем «что произошло» в приложении.
        


        

          «An action is a plain JavaScript object that has a type field. You can think
          of an action as an event that describes something that happened in the application.
          The type field should be a string that gives this action a descriptive name,
          like "todos/todoAdded".»
          
— Redux Fundamentals, Part 2
        


        
Обязательное поле: type

        

          Поле type — строка, дающая экшену уникальное описательное имя.
          Принятое соглашение — формат "домен/событие":
        


        

// Формат: "домен/чтоПроизошло"
{ type: 'counter/incremented' }
{ type: 'todos/todoAdded' }
{ type: 'filters/statusChanged' }
        


        
Дополнительные данные: payload

        

          Экшен может содержать дополнительную информацию о том, что произошло.
          По соглашению, эти данные помещаются в поле payload:
        


        

const addTodoAction = {
  type: 'todos/todoAdded',
  payload: 'Купить молоко'
}

const addByAmount = {
  type: 'counter/addAmount',
  payload: 5
}
        


        
Как это работает в этом примере

        

import { createStore } from 'redux'

// Reducer — обрабатывает экшены и возвращает новый state
function counterReducer(state = { value: 0 }, action) {
  switch (action.type) {
    case 'counter/incremented':
      return { ...state, value: state.value + 1 }
    case 'counter/decremented':
      return { ...state, value: state.value - 1 }
    case 'counter/addAmount':
      return { ...state, value: state.value + action.payload }
    case 'counter/reset':
      return { value: 0 }
    default:
      return state  // ВАЖНО: неизвестные экшены — возвращаем state без изменений
  }
}

// Создаём store
const store = createStore(counterReducer)

// Dispatch — единственный способ изменить state
store.dispatch({ type: 'counter/incremented' })
store.dispatch({ type: 'counter/addAmount', payload: 5 })
        


        

          Обратите внимание на DevTools-панель выше — она показывает каждый
          отправленный экшен, текущий state и разницу (diff) между предыдущим и новым состоянием.
          Это аналог расширения Redux DevTools для Chrome, встроенный прямо в страницу.
