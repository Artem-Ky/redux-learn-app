Теория — Стрелочные Action Creator

      

        
Обычная функция vs Стрелочная

        

          Action creator — это функция, возвращающая объект. Её можно записать двумя способами.
          Результат одинаковый — разница только в синтаксисе:
        


        

// ═══ Обычная функция (function declaration) ═══
function increment() {
  return { type: 'counter/incremented' }
}

function addAmount(amount: number) {
  return { type: 'counter/addAmount', payload: amount }
}
        


        

// ═══ Стрелочная функция (arrow function) ═══
const increment = () => ({
  type: 'counter/incremented'
})

const addAmount = (amount: number) => ({
  type: 'counter/addAmount',
  payload: amount
})
        


        
Неявный return и скобки

        

          Ключевая особенность стрелочных функций — implicit return
          (неявный возврат). Но есть важный нюанс: объектные литералы {}
          нужно оборачивать в круглые скобки (),
          иначе JavaScript воспримет фигурные скобки как тело функции:
        


        

// ❌ ОШИБКА — JS думает, что {} — тело функции
const increment = () => { type: 'counter/incremented' }
// Вернёт undefined!

// ✅ ПРАВИЛЬНО — скобки говорят, что это объект
const increment = () => ({ type: 'counter/incremented' })
        


        

          Не забывайте круглые скобки вокруг объектного литерала
          при использовании implicit return в стрелочной функции:
          () => ({ ... }), а не () => { ... }
        


        
Когда что использовать?

        
    
          
  • Стрелочная — для простых action creator'ов (1 строка, прямой return объекта)
    • 
          
  • Обычная — когда внутри есть логика (генерация ID, проверки, подготовка данных)
    • 
        


        

// Стрелочная — коротко и ясно для простых случаев
const increment = () => ({ type: 'counter/incremented' })
const decrement = () => ({ type: 'counter/decremented' })

// Обычная — когда нужна логика
let nextId = 1
function todoAdded(text: string) {
  const id = nextId++
  return {
    type: 'todos/todoAdded',
    payload: { id, text }
  }
}
        


        

          В сообществе Redux стрелочные функции — самый популярный стиль для простых
          action creator'ов. Они компактные и хорошо читаются. Оба варианта
          функционально идентичны — выбор за вами.
        


        
Ключевые моменты

        
    
          
  • Стрелочная функция с implicit return: const fn = () => ({ ... })
    • 
          
  • Обязательно оборачивайте объект в ()
    • 
          
  • Оба синтаксиса создают идентичные объекты экшенов
    • 
          
  • Стрелочные — для простых, обычные — для сложных creator'ов
