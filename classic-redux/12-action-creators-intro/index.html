Теория — Что такое Action Creator?

      

        
Определение

        

          Action Creator — это функция, которая создаёт и возвращает action-объект.
          Вместо того чтобы каждый раз вручную писать объект { type: '...' },
          вы вызываете функцию.
        


        

          «An action creator is a function that creates and returns an action object.
          We typically use these so we don't have to write the action object by hand every time.»
          
— Redux Fundamentals, Part 7
        


        
Простой action creator

        

// Action creator — просто функция, возвращающая action
function increment(): CounterAction {
  return { type: 'counter/incremented' }
}

function decrement(): CounterAction {
  return { type: 'counter/decremented' }
}

function reset(): CounterAction {
  return { type: 'counter/reset' }
}

// Использование
store.dispatch(increment())  // вместо { type: 'counter/incremented' }
store.dispatch(decrement())  // вместо { type: 'counter/decremented' }
        


        
Зачем использовать Action Creators?

        
    
          
  • DRY — структура action описана в одном месте, не дублируется
    • 
          
  • Инкапсуляция — dispatch-код не знает деталей реализации action
    • 
          
  • Защита от ошибок — тип экшена задаётся только в action creator
    • 
          
  • Удобство рефакторинга — при изменении структуры action, правите только одну функцию
    • 
          
  • Тестируемость — action creator легко тестировать как чистую функцию
    • 
        


        
Без action creator vs с ним

        

// ❌ Без action creator — дублирование
// файл button.ts
store.dispatch({ type: 'counter/incremented' })
// файл header.ts
store.dispatch({ type: 'counter/incremented' })
// файл sidebar.ts
store.dispatch({ type: 'counter/incremented' })

// ✔ С action creator — единый источник
// файл button.ts
store.dispatch(increment())
// файл header.ts
store.dispatch(increment())
// файл sidebar.ts
store.dispatch(increment())
        


        

          Обратите внимание: оба подхода в демо выше дают абсолютно одинаковый результат.
          Action creator — это просто удобная обёртка, которая не меняет поведение Redux.
          В DevTools экшены от обоих подходов выглядят идентично.
        


        
Action Creator + константы

        

          Часто action creators комбинируют с константами из предыдущего урока:
        


        

const INCREMENTED = 'counter/incremented'

function increment() {
  return { type: INCREMENTED }
}
        


        

          В классическом Redux action creators нужно писать вручную. В Redux Toolkit
          createSlice генерирует их автоматически — это одно из главных
          преимуществ RTK.
