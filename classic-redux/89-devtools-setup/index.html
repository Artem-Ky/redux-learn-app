Теория — Подключение Redux DevTools

      

        
Расширение для браузера

        

          Redux DevTools — это расширение для Chrome и Firefox, которое позволяет
          инспектировать каждый экшен, просматривать состояние, видеть diff между
          состояниями и путешествовать во времени.
        

        
    
          
  • Chrome Web Store — Redux DevTools
    • 
          
  • Firefox Add-ons — Redux DevTools
    • 
        


        
Подключение через enhancer

        

          После установки расширения в window появляется глобальный
          объект __REDUX_DEVTOOLS_EXTENSION__. Вызывая его, мы получаем
          store enhancer:
        


        

const enhancer = window.__REDUX_DEVTOOLS_EXTENSION__?.() || (f => f)

const store = createStore(reducer, enhancer)
        


        
Пакет redux-devtools-extension

        

          Для более удобного подключения (особенно вместе с middleware) существует
          npm-пакет redux-devtools-extension:
        


        

import { composeWithDevTools } from 'redux-devtools-extension'

const store = createStore(
  reducer,
  composeWithDevTools(applyMiddleware(thunk))
)
        


        

          «Redux was specifically designed to make it easier to trace when, where,
          why, and how your state has changed. Redux DevTools allow you to see the
          history of dispatched actions, the contents of each action, the final state
          after an action was dispatched, and the diff in the state after an action.»
          
— Redux Fundamentals, Part 4: Store — Redux DevTools
        


        
Что видно в DevTools

        

          Расширение Redux DevTools показывает:
        

        
    
          
  • Actions — список всех отправленных экшенов с типом и payload
    • 
          
  • State — полное дерево состояния после каждого экшена
    • 
          
  • Diff — что именно изменилось в состоянии
    • 
          
  • Time Travel — возможность «перемотать» к любому состоянию
    • 
          
  • Dispatch — отправка экшенов прямо из DevTools
    • 
        


        

          Наша встроенная DevTools-панель работает по тому же принципу, что и расширение:
          перехватывает dispatch, сохраняет снимки состояния «до» и «после»,
          и показывает историю экшенов. Откройте расширение Chrome — вы увидите те же данные.
        


        

          window.__REDUX_DEVTOOLS_EXTENSION__ доступен только когда расширение
          установлено. Всегда используйте optional chaining (?.) или fallback
          || (f => f), чтобы приложение работало без расширения.
        


        
Ключевые моменты

        
    
          
  • Установите расширение Redux DevTools для Chrome или Firefox
    • 
          
  • window.__REDUX_DEVTOOLS_EXTENSION__?.() — enhancer для store
    • 
          
  • composeWithDevTools — удобное подключение вместе с middleware
    • 
          
  • DevTools показывают: actions, state, diff, time travel
    • 
          
  • Не забывайте fallback для продакшена или без расширения
    •
