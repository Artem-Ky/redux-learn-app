Итоги курса

Что ты теперь умеешь

  • Настраивать configureStore с middleware и enhancers — и понимать, что каждый делает.
  • Писать createSlice во всех формах (basic, prepare, extraReducers, creator callback).
  • Использовать createAsyncThunk с thunkAPI, rejectWithValue, condition, signal.
  • Нормализовать state через createEntityAdapter.
  • Писать реактивную логику через listenerMiddleware.
  • Строить типизированный data fetching на RTK Query с тегами, optimistic updates, streaming, infinite queries, code splitting.
  • Отлаживать через Redux DevTools, понимать внутренности (cacheKey, state.api, lifecycle).
  • Избегать топ-15 подводных камней (урок 105).
