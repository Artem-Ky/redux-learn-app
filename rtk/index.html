Redux Toolkit — Полный курс
К проекту
RTK + RTK Query — глубокое погружение
Курс из 106 интерактивных уроков по официальному инструменту Redux. От первой configureStore до тонкостей createApi с tags, optimistic updates и streaming. Каждый урок — живой пример + теория с цитатами из официальной документации и фрагментами исходников @reduxjs/toolkit.
Уроков: 106 Секций: 13 (A–M) Стек: React 19 + TS + Vite Версия: @reduxjs/toolkit 2.5+
